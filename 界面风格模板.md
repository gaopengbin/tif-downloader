# 界面风格模板 (GeoAI Pro UI Style Guide)

本文档定义了 GeoAI Pro 前端界面的设计规范、配色系统及核心组件样式。开发新功能时，请严格遵循本指南以保持界面一致性。

## 1. 核心设计理念 (Design Philosophy)

*   **沉浸式地理空间 (Immersive Geospatial)**: 始终保持地图（MapView）全屏显示，作为应用的基础底座。
*   **悬浮式交互 (Floating Interactions)**: 操作面板采用白色半透明材质悬浮于地图之上，不遮挡地图视野。
*   **现代简洁 (Modern & Clean)**: 使用大圆角、柔和阴影和清晰的层级关系，避免过度装饰。

## 2. 颜色系统 (Color System)

基于 Tailwind CSS 配置，项目核心配色如下：

### 主色调 (Primary Palette)
用于品牌标识、主要行动点（CTA）、选中状态。
*   **Primary Blue**: `#0052cc` (默认主色)
*   **Primary Hover**: `#0047b3` (按钮悬停)
*   **Primary Light**: `rgba(0, 82, 204, 0.1)` (浅色背景衬托，如选中项背景)

### 状态色 (Status Colors)
用于反馈系统状态。
*   **Success**: `#36b37e` (成功/就绪)
*   **Error**: `#ff5630` (失败/错误)
*   **Warning**: `#ff9800` (警告/加载中)

### 中性色 (Neutrals)
*   **Panel Background**: `rgba(255, 255, 255, 0.95)` (面板背景)
*   **Text Primary**: `text-gray-800` (正文)
*   **Text Secondary**: `text-gray-500` (次要信息、说明)
*   **Border**: `border-gray-200` (分割线、边框)

## 3. 全局样式类 (Global Utility Classes)

项目在 `src/styles/main.css` 中定义了以下复用类，请优先使用：

### 3.1 面板 (Panels)
所有悬浮在地图上的容器应使用 `.panel` 类。
```css
.panel {
  @apply bg-white/95 backdrop-blur-sm rounded-lg shadow-panel;
}
/* shadow-panel: 0 4px 12px rgba(0,0,0,0.15) */
/* backdrop-blur: 10px */
```

### 3.2 按钮 (Buttons)
**主按钮 (Primary Action)**
用于提交表单、开始任务等主要操作。
```css
.btn-primary {
  @apply w-full py-3 px-4 bg-primary text-white border-none rounded-lg font-semibold cursor-pointer
         flex justify-center items-center gap-2 transition-all duration-200
         hover:bg-primary-hover hover:-translate-y-0.5 hover:shadow-hover;
}
```

**次级按钮 (Secondary Action)**
用于取消、辅助操作或多选标签。
```css
.btn-secondary {
  @apply py-2 px-4 bg-white border border-gray-200 rounded-lg text-gray-700 font-medium cursor-pointer
         flex items-center justify-center gap-2 transition-all duration-200
         hover:bg-gray-50 hover:border-primary hover:text-primary;
}
```

### 3.3 表单控件 (Form Controls)
**输入框 (.input)**
```css
.input {
  @apply w-full py-2 px-3 border border-gray-200 rounded text-sm
         focus:outline-none focus:ring-2 focus:ring-primary/20 focus:border-primary;
}
```

**标签 (.label)**
```css
.label {
  @apply block text-sm font-medium text-gray-700 mb-1.5;
}
```

**文件上传区 (.file-upload)**
大块的可点击区域，支持拖拽感。
```css
.file-upload {
  @apply flex items-center gap-3 p-3 border border-dashed border-primary rounded-lg
         bg-primary-light cursor-pointer text-sm text-primary transition-all duration-200
         hover:bg-primary/10;
}
/* 次级上传样式 */
.file-upload.secondary {
  @apply border-gray-400 text-gray-500 bg-transparent;
}
```

**开关 (.toggle-switch)**
自定义的 iOS 风格开关。

## 4. 布局规范 (Layout Specifications)

### 侧边栏 (Sidebar)
*   **位置**: 左上角悬浮 (`absolute top-5 left-5`)
*   **尺寸**: 宽度固定 `w-[360px]`
*   **高度**: 最大高度 `max-h-[calc(100vh-40px)]`，内部内容滚动
*   **层级**: `z-[1000]`

### 结果面板 (Result Panel)
*   **位置**: 右上角悬浮
*   **风格**: 与侧边栏一致，保持对称性

## 5. Mapbox 样式覆盖
为了保持风格统一，Mapbox 原生控件已被重写：
*   **圆角**: 所有控件按钮统一为 `rounded-lg`
*   **阴影**: 统一使用 `shadow-panel`
*   **背景**: 去除纯白，使用 `bg-white/95` 半透明

## 6. 图标与字体
*   **字体**: 系统默认 sans-serif 字体栈 (`-apple-system`, `BlinkMacSystemFont`...)
*   **图标库**: FontAwesome 6 Pro (Solid & Regular)
    *   示例: `<i class="fa-solid fa-layer-group"></i>`

## 7. 动画效果
*   **加载中**: `.animate-spin` (旋转)
*   **进入**: `.animate-fadeIn` (0.3s 上浮淡入)
*   **交互反馈**: 按钮 Hover 时有轻微上浮 (`-translate-y-0.5`)
